Some widget extensions and common classes for improve developer experience.

## Features

# Widget extensions:
* paddings
* margins
* Expanded
* GestureDetector
* SliverToBoxAdapter
* MouseRegion
* SingleChildScrollView
* Positioned
* Positioned.fill

# DateTime extensions:
* isLeapYear
* daysInMonth
* nextDay
* isAfter
* isBefore
* isSameDay
* removeTime
* firstDayOfWeek
* lastDayOfWeek
* lastDayOfMonth

# Text utilites:
* getNoun (for plurals)
* parseLink

# Text formatters:
* UpperCaseTextFormatter


# Classes:
* MultitoolBlocObserver
* Either (aka Either from dartz library)
